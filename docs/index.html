Install in one line

Nothing to configure.

No Python, no pip — uv brings its own. Chrome and Chrome DevTools MCP auto-register with your AI tools.

$ uvx fagun init
# installs browser + Fagun MCP + Chrome DevTools MCP
# Chrome DevTools usage stats disabled in generated config
$ uvx fagun install claude-code
# or target one tool
